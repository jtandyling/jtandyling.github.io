CPMP Website Launch

[DRAFT] January 17, 2026 by James Tandy

I have restructured my personal website so that the Colonial Poqom Manuscript Project now has its own self-contained directory. I also made some improvements to the site's visual design. The original "Colonial Poqom resources" page has become the Manuscripts page on this site. So far, all the content is the same: the Manuscripts page includes links to the Excel database for the project and all available transcriptions. More updates are planned for the near future, to improve organization and ease of access. Some new transcriptions are also in the pipeline - stay tuned!
